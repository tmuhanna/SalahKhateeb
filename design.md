# Law Firm Website - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary Colors**: Deep navy blue (#1a365d), charcoal gray (#2d3748), warm white (#fafafa)
- **Accent Colors**: Gold (#d4af37), soft blue (#4299e1)
- **Text Colors**: Dark gray (#1a202c) for headings, medium gray (#4a5568) for body text
- **Background**: Clean white (#ffffff) with subtle gray sections (#f7fafc)

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings and hero text
- **Body Font**: "Inter" - Clean, modern sans-serif for readability
- **Accent Font**: "Crimson Text" - Sophisticated serif for quotes and emphasis

### Visual Language
- **Professional Authority**: Clean lines, sophisticated color palette, elegant typography
- **Trust & Credibility**: Consistent spacing, professional imagery, refined animations
- **Modern Elegance**: Subtle shadows, smooth transitions, premium feel
- **Accessibility**: High contrast ratios, readable fonts, clear hierarchy

## Visual Effects & Animation

### Used Libraries
- **Anime.js**: Smooth scroll animations, section reveals, text effects
- **Typed.js**: Typewriter effect for hero section tagline
- **Splitting.js**: Text animation effects for headings
- **ECharts.js**: Data visualization for firm statistics
- **Splide.js**: Team member carousel and testimonial slider

### Effect Implementation
- **Hero Section**: Typewriter animation for main tagline, subtle parallax background
- **Section Reveals**: Fade-in animations triggered at 50% viewport entry
- **Service Cards**: 3D hover effects with shadow expansion and slight rotation
- **Team Profiles**: Overlay animations revealing member specializations
- **Contact Form**: Real-time validation with smooth color transitions

### Header Effect
- **Navigation Bar**: Fixed position with subtle backdrop blur on scroll
- **Smooth Scrolling**: Anime.js powered scroll-to-section navigation
- **Active States**: Elegant underline animations for current section

### Styling Approach
- **Consistent Spacing**: 8px grid system throughout
- **Subtle Shadows**: Soft drop shadows for depth without distraction
- **Hover States**: Sophisticated transitions with 300ms duration
- **Mobile-First**: Responsive design with breakpoints at 768px and 1024px
- **Professional Imagery**: High-quality office and team photography

## Section-Specific Design

### Hero Section
- Full-height hero with professional office background
- Centered content with animated tagline and call-to-action
- Subtle overlay for text readability

### About Section
- Split layout with text and professional imagery
- Animated statistics with ECharts visualization
- Timeline of firm achievements

### Services Section
- Grid layout with interactive service cards
- Hover effects revealing detailed descriptions
- Icon animations and color transitions

### Team Section
- Professional headshots with overlay information
- Carousel navigation for mobile devices
- Specialization tags with smooth animations

### Contact Section
- Clean form design with floating labels
- Interactive validation with visual feedback
- Office location with subtle map integration
